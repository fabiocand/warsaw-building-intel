# Warsaw Building Intelligence (Gemini — Free)

Powered by Google Gemini 2.0 Flash with Google Search grounding. Free tier: 1500 requests/day.

## Get a free Gemini API key

1. Go to https://aistudio.google.com
2. Click "Get API key" → "Create API key"
3. Copy the key

## Deploy to Vercel

1. Upload this folder to a GitHub repo
2. Go to vercel.com → New Project → import the repo
3. Under Environment Variables add:
   - Key: `GEMINI_API_KEY`
   - Value: your key from step above
4. Click Deploy

Your site is live — no billing required.
